# Socket Designation

---

## 🚀 Features

- ✅ Register and login system  
- 📡 Real-time communication using sockets  
- 👥 Create and join chat groups  
- 💬 Broadcast and group messaging  
- 🌐 Supports multiple client connections (via `0.0.0.0` binding)
